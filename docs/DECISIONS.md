# Admin Service — Архитектурные решения

## ADR-001: Порт 8097 вместо 8095

**Контекст:** В дизайн-документе указан порт 8095, но он уже занят Analytics Service.
**Решение:** Используем порт 8097.
**Последствия:** Нужно обновить все ссылки на порт в документации.

## ADR-002: Redis для feature flags

**Контекст:** Feature flags должны применяться мгновенно ко всем сервисам.
**Решение:** Хранить в Redis, читать без кэша где критично.
**Последствия:** Зависимость от Redis для конфигурации.

## ADR-003: Только super_admin

**Контекст:** Админ-панель содержит чувствительные операции.
**Решение:** Проверка роли super_admin в каждом запросе (middleware).
**Последствия:** Требуется Auth Service для JWT валидации.
